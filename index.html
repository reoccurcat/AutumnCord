What's AutumnCord?

AutumnCord is a fully open sourced Discord bot. The bot has a lot of stuff, such as image commands, moderation commands, fun commands, utility commands, and more!

  • Add AutumnCord

Features

  • Fully Open Source

    The code is completely open for anyone to see and make edits to. The license used is the GNU GPL v3 license.

  • Secure

    The bot is hosted locally on reoccurcat's servers. Not some random VPS in some random country.

  • Free. Forever.

    A supporters version may come along, but the base will stay free. No premium nags. At all.

What makes this bot special?

  • v1.0 Updates come fast and quickly, I'm working hard
  • >10 Commands (and more being added frequently)
  • ~100ms The ping is pretty good on my crap internet
  • Secure Nothing personal leaves my server
  • Cats Who doesn't love cats? (or dogs)

Besides all of the above, we also have a really strong support team (well, mostly consisting of me, but I coded the whole bot, so I should know what I'm doing) and if there's an error I'll work to fix it with you! And, to be honest, I'm a young coder. I pour my heart and soul into this bot and it would be amazing for you to give it a shot.
